Welcome to Documentation for EMCCD QE Analysis!
===================================

**EMCCD QE Analysis** is a Python library for the reduction of Quantum Efficiency measurements for EMCCDs taken using the UV/Vis Monochramtor setup at Hamden UV/Vis Detector lab. 

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
